# Offline Railway Chatbot

A web-based railway information chatbot that works offline by using a local database. Users can check train schedules, fares, station facilities, and routes without internet connection.

## About
Independently designed and developed — managing all 
phases from database design to frontend integration.

## Tech Stack
- Python (Flask)
- JavaScript
- HTML, CSS
- SQLite (preloaded local database)
- REST APIs

## Features
- Works completely offline without internet
- Provides train fare and schedule information
- ~85% query accuracy across 500+ preloaded records
- Fast response using Flask REST API backend
- Designed for users with limited internet access

## How to Run
1. Clone this repository
2. Install dependencies: pip install -r requirements.txt
3. Run the app: python app.py
4. Open your browser and go to: http://127.0.0.1:5000
