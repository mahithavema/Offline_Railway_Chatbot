# Offline Railway Chatbot

A web-based railway information chatbot that works offline by using a local database. Users can check train schedules, fares, station facilities, and routes without internet connection.

## Features
- User registration and login
- Train search by source and destination
- Train fare details
- Train route/stops information
- Station facilities lookup
- Simple chat interface

## Tech Stack
- Python Flask
- MySQL Database
- HTML, CSS
